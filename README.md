# FitFlow — Fitness Planner

FitFlow is a modern fitness planner web application that generates personalized workout and nutrition plans based on user goals, experience level, and lifestyle preferences.

Built with Next.js, Tailwind CSS, and OpenAI integration.

---

## Features

### AI Workout Planning

* Personalized training plans
* 7-day workout structure
* Strength / hypertrophy / fat loss logic
* Progressive overload recommendations
* Rest & recovery guidance
* Warm-up and cooldown suggestions
* Beginner / intermediate / advanced adaptation
* Compound & isolation exercise labeling

### AI Nutrition Planning

* Personalized meal generation
* Daily calorie & macro calculations
* Protein / carbs / fat tracking
* Hydration reminders
* Grocery suggestions
* Meal timing recommendations
* High-protein & muscle gain support

### Premium UI/UX

* Modern dark-mode design
* Mobile responsive layout
* Interactive dashboard
* PDF export support
* Coach insights
* Clean card-based architecture

## Installation

Clone the repository:

```bash
git clone https://github.com/mehmetkeless00/fitai-ai-fitness-planner.git
```

Navigate into the project:

```bash
cd fitai-ai-fitness-planner
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

## Environment Variables

Create a `.env.local` file:

```env
OPENAI_API_KEY=your_api_key_here
```
